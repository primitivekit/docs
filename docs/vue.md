# Vue Components

PrimitiveKit for Vue is coming soon! 🚀

## What to Expect

We're working on bringing the same great experience to Vue developers:

- ✅ **38 Components** - All components from React version
- ✅ **Vue 3** - Built for Vue 3 Composition API
- ✅ **TypeScript** - Full type definitions
- ✅ **Accessible** - WCAG 2.1 AA compliant
- ✅ **Customizable** - Same 600+ design tokens
- ✅ **SSR Ready** - Works with Nuxt 3

## Planned Features

### Composition API

```vue
<script setup>
import { Button } from '@primitivekit/vue';
</script>

<template>
  <Button variant="primary">
    Click me
  </Button>
</template>
```

### TypeScript Support

```vue
<script setup lang="ts">
import { Button, type ButtonProps } from '@primitivekit/vue';

const props = defineProps<ButtonProps>();
</script>
```

### Nuxt 3 Integration

```vue
<!-- pages/index.vue -->
<script setup>
import { Button } from '@primitivekit/vue';
</script>

<template>
  <div>
    <Button>Click me</Button>
  </div>
</template>
```
