# Test Page

This is a test page to verify the framework switcher is working.

## Instructions

1. Look at the **bottom-right corner** of the page
2. You should see a **blue floating button** that says "Frameworks"
3. Click it to open the mega menu
4. The menu should show all 6 frameworks with logos

## Troubleshooting

If you don't see the button:
- Try refreshing the page (Cmd+R or Ctrl+R)
- Try hard refresh (Cmd+Shift+R or Ctrl+Shift+R)
- Check if JavaScript is enabled
- Check browser console for errors

## What You Should See

- **Floating Button**: Blue pill-shaped button in bottom-right
- **Text**: "Frameworks" with icons
- **On Click**: Mega menu appears above the button
- **Menu Content**: 6 framework cards with logos
