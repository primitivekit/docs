# Showcase

Discover amazing projects built with PrimitiveKit. See how developers are using our component library to create beautiful, accessible user interfaces.

## Featured Projects

We're collecting examples of projects using PrimitiveKit. If you've built something with PrimitiveKit, we'd love to feature it here!

### How to Submit Your Project

1. Create a pull request on our [GitHub repository](https://github.com/primitivekit/react)
2. Include:
   - Project name and description
   - Screenshot or demo link
   - Which frameworks you're using
   - Any interesting customizations
