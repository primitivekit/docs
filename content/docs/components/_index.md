PrimitiveKit provides 38 production-ready components for building modern user interfaces. All components are fully accessible, customizable, and work with React, Vue, Svelte, Angular, and HTML/CSS.

## Form Components

Essential form controls for user input.

- [Button](/docs/components/button) - Versatile button with multiple variants
- Input - Text input with validation
- Textarea - Multi-line text input
- Select - Dropdown selection
- Checkbox - Multiple selection control
- Radio - Single selection control
- Switch - Toggle switch
- Slider - Range value selector
- Upload - File upload with drag-and-drop

## Layout Components

Components for structuring your UI.

- Card - Content container
- Container - Max-width wrapper
- Grid - Responsive grid system
- Stack - Flex layout helper
- Divider - Visual separator
- Space - Spacing utility

## Feedback Components

Components for user feedback and status.

- Badge - Status indicator
- Spinner - Loading animation
- Alert - Alert messages
- Progress - Progress indicator
- Skeleton - Loading placeholder

## Navigation Components

Components for site navigation.

- Link - Navigation link
- Tabs - Tabbed content
- Breadcrumb - Navigation trail
- Steps - Step indicator
- Pagination - Page navigation

## Overlay Components

Modal and floating components.

- Modal - Dialog overlay
- Tooltip - Hover tooltip
- Popover - Floating content
- Dropdown - Dropdown menu
- Drawer - Sliding panel

## Data Display

Components for displaying data.

- Avatar - User profile picture
- Tag - Label or category
- Empty - Empty state
- Image - Enhanced image
- Collapse - Collapsible panels
- Table - Data table

## General

Utility components.

- Typography - Text styling
- Menu - Navigation menu

## Getting Started

Each component page includes:

- ✅ Installation instructions
- ✅ Examples for all frameworks (React, Vue, Svelte, Angular, HTML)
- ✅ Props/API documentation

## Framework Support

All components work with:

- **React** - Full component library
- **Vue** - CSS classes with design tokens
- **Svelte** - CSS classes with design tokens
- **Angular** - CSS classes with design tokens
- **HTML/CSS** - Pure HTML with CSS variables

## Customization

Every component is fully customizable via CSS variables. See the [Customization Guide](/docs/customization) for details.

## Accessibility

All components follow WCAG 2.1 AA standards. See the [Accessibility Guide](/docs/accessibility) for details.
